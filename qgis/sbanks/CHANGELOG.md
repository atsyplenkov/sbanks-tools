### Added
- Initial release of sbanks QGIS plugin
- Savitzky-Golay filter algorithm for geometry smoothing with anti-hook extrapolation
- Whittaker-Eilers filter algorithm for geometry smoothing with distance-aware smoothing
- Support for (MULTI)LINESTRING and (MULTI)POLYGON geometries
